# Examples

Run examples from package root.

Core:
```bash
dart run example/core_example.dart
```

HTTP adapter (`package:http`, Chopper compatible):
```bash
dart run example/http_adapter_example.dart
```

Dio adapter:
```bash
dart run example/dio_adapter_example.dart
```

Local server adapter:
```bash
dart run example/server_example.dart
```
